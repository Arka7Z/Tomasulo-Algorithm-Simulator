To Run the code:
------------------------------------------------------------
Compile the file by running: g++ -std=c++11 simulator.cpp
To generate output from an input file, run:    ./a.out < input_file_name. For example, ./a.out< ip.txt; after compiliing.

Two sample input and output pairs( ip.txt -> op.txt & ip1.txt->op2.txt) have been provided.
